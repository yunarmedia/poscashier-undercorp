# poscashier-undercorp
Solusi kasir all-in-one untuk manajemen bisnis Anda. Fitur lengkap: sinkronisasi menu cloud via Firebase, manajemen stok real-time, pencatatan pengeluaran, laporan omzet otomatis, hingga cetak struk Bluetooth ganda (kasir &amp; dapur). Efisien, responsif di HP/tablet/PC, dan dirancang khusus untuk operasional bisnis yang cepat, rapi, dan modern.
